# 🎯 FocusFlow – Smart Productivity Tracker

## 🚀 Overview
FocusFlow is a simple and efficient web application that helps users track their focus sessions and monitor distractions. It is designed to improve productivity and time management.

## ✨ Features
- ⏱ Start and stop focus sessions
- 📱 Track distractions (Instagram, YouTube, etc.)
- 📊 View total sessions and focus time
- 💾 Data stored in browser (no backend required)
- 🎨 Clean and modern UI

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Click **Start** to begin a session
2. Click **Stop** to end session
3. Add distractions when you lose focus
4. View your productivity stats
